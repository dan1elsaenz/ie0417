Gestor de Tareas Personales
===========================

Bienvenido a la documentación oficial del **Gestor de Tareas del Laboratorio 2** del curso *IE-0417*.

Este proyecto está desarrollado en Python y permite la gestión de proyectos personales y sus tareas
asociadas. La aplicación ofrece una interfaz en consola que permite crear, listar, completar y verificar tareas.
